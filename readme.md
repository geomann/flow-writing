# flow-writing是什么

flow-writing是一个受到flowstate启发的自用脚本，通过netlify部署在网页上，可以通过电脑和手机随时随地访问的“呕吐式”写作小工具，旨在让自己大脑不假思索地在规定时间内连续写东西，让自己的想法不受审查地流露出来。

一定程度上，这种写作的时候人是沉浸在心流里的，也有一种通过写作来进入冥想状态的企图。
